Week 1: 8/21 - 8/27

Last Updated 2017-08-20 19:04:05

Monday: Gearing up

Watch the Introduction Video in Blackboard learn. In class attendance is optional.

Goals:

  • See an eclipse
  • Understand how the class is run, common acronyms (OI/CN/DC/RAT/BBL/PDS)
  • Understand how to navigate the course website
  • Understand how learning is your first responsibility in this class, and how it is hard.
  • Understand the concept of metacognition and how being aware of your learning can improve your chances of success
  • Understand what it takes to get an A in this course, and what it takes “just to pass”
  • Get connected to online accounts and resources
  • Start learning R using Data Camp

Prepare

  • Get your eclipse viewing materials prepared.
  • Read the Syllabus
  • Complete the OH Survey
  • Bookmark this class website
  • Purchase the Course Notes (CN) packet (required) from Chico Packet Pro ($18.23)
    • Chico Packet Pro store will be open from:
      • Friday, August 18 - Wednesday, September 6
      • Mon-Thu 10-6, Fri 10-5, Sat 8/19 + 8/26 from 10-2
    • Chico Packet Pro is located at 230 Salem Street and our entrance is on the city parking lot side of the building. We accept cash, debit and credit cards.
    • Online at https://squareup.com/store/chico-packet-pro/item/math-course-notes-prof-donatello
  • Join the class Data Camp group Use link in email
  • Join the Chico R Users Google Group - This is our discussion forum to ask (and answer) questions outside of class.
  • Create a MATH315 folder on your computer - all course related files go in here.
  • Download the color PDF version of the course notes for reference - This is not a replacement for the print version.
  • Download or purchase Open Intro (OI) Textbook

In class

Open to questions / help getting your preparation and HW tasks done before Wednesday.

Assignments
  • Complete the Intro to R Data Camp Assignment (Assigned to you in the Data Camp Classroom). If you have completed any R Workshop prior to class you probably don’t need to do this.
  • Complete the Intro to R BBL quiz (follows from the data camp assignment).
  • Watch Passion Driven Statistics (PDS) Video 1.
  • Read Our path to better science in less time using open data science tools (Nature: Ecology and Evolution)
  • Read Learning - Your First Job
  • Read Metacognition and academic achievement
    • Take the Metacognition Awareness Inventory
    • Complete the Metacognition Pre Assessment Google Survey

Wednesday: Starting R

Goals:

  • Create support groups
  • Get your computer prepared for data analysis
  • Produce a reproducible document using R Markdown.
  • Understand how R markdown will be used to submit homework
  • Learn what data set you will be exploring throughout the class

Prepare

  • Follow the Software overview blog post to install necessary programs onto your own computer
  • Right click and “save link as” to download the HW 1 [RMD Template] and [PDF] files into your class folder.
  • Download and review the Add Health Wave IV PDF codebook and pick 3-5 sections you are interested in exploring.

In class

  • Ask questions about class logistics and structure
  • Create support groups.
  • The data analysis life cycle
  • Introduction to the Add Health data
  • Introduction to R Markdown. Or, “How do you turn in homework.”
Assignments
  • R Markdown Test file
  • HW1 - Upload knitted WORD or PDF file to BBL

Friday: Data architecture

Goals:

  • Learn how data is stored
  • Identify the different data types
  • Learn about the data set on Adolescents you will be using
  • Learn how to read a data dictionary (codebook)

Prepare

  • Watch PDS Video 2
  • Read Course Notes (CN) Section 2.1 and Open Intro Textbook (OI) Section 1.2. Write answers to questions and examples in the CN.
  • Download the medical records PDF data from the raw data page.

In class

  • Reading assessment on class logistics, metacognition and data collection
  • Form analysis pairs
  • Share answers to questions in CN
  • Medical records assignment [Instructions]
Assignments
  • PS 2.2: Identifying Data
  • Read the Week 2 overview so you know how to prepare for Monday. (This is a recurring theme. Always look ahead to see how you need to manage your time.)
